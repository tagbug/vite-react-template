# vite-react-template

## Getting Start

Prerequisites:

- [Node](https://nodejs.org/en/) version >= v16
- [pnpm](https://pnpm.io/) version >= v7.
- (Recommend) Familiar with Git.

Install required packages:

```bash
$ pnpm i
```

To running in dev-env:

```bash
$ pnpm dev
```

To build the product version:

```bash
$ pnpm build
```
